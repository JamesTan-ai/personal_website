### 1. Download
```bash
git clone https://github.com/sober-baby/phone_chain.git

```

### 2. Install
```bash
cd website
npm install
```

### 3. Run
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
